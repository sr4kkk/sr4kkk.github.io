Instalação

Como instalar no VS Code

Você pode instalar pelo Marketplace ou pelo GitHub.

Opção 1 — Marketplace (se publicado)

  1. Abra o VS Code.
  2. Vá em Extensions (Ctrl+Shift+X).
  3. Pesquise por Graphite Red.
  4. Clique em Install e selecione em Preferences → Color Theme.

Opção 2 — Pelo GitHub

  1. Abra o repositório no GitHub e baixe como ZIP (ou clone).
  2. Extraia para a pasta de extensões do VS Code:
Windows: %USERPROFILE%\.vscode\extensions
macOS: ~/.vscode/extensions
Linux: ~/.vscode/extensions

Depois reinicie o VS Code e selecione em Preferences → Color Theme.

Ativar e alternar

  1. Abra a paleta: Ctrl+Shift+P
  2. Digite Color Theme
  3. Escolha Graphite Red (Dark) ou Graphite Red (Light)

Se o tema não aparecer, confira o FAQ abaixo.

FAQ

Perguntas frequentes

Dúvidas rápidas sobre instalação e uso.

Instalei e o tema não apareceu. O que fazer?

Reinicie o VS Code e abra Preferences → Color Theme. Se instalou via pasta, confirme se o diretório do tema ficou dentro de .vscode/extensions e se não está “aninhado” em subpastas.

Como alternar entre Dark e Light?

Abra a paleta (Ctrl+Shift+P), digite Color Theme e selecione Graphite Red (Dark) ou Graphite Red (Light).
